Task Hub - таск-трекер для управления задачами и проектами. Подходит компаниям до 20 пользователей. Бэкенд с использованием Supabase.

Реализован следующий функционал:

- Авторизация
- Графики проектов за месяц или год
- Eсть прелоадер и обработка ошибок с сервера
- Фильтрация и сортировка задач
- Чат
- Прогресс бар
- Модальные окна редактирования и добавления задач и валидация

Технологии которые были использованы:
Next, SCSS, Zod, React Hook Form, Tanstack Query, Clsx, MobX
